# Changelog

This project uses [GitHub Releases](https://github.com/godaddy/asherah-ffi/releases)
to track changes. Each release includes native libraries for all platforms and
triggers automated publishing to npm, PyPI, NuGet, Maven, and RubyGems.

See the [releases page](https://github.com/godaddy/asherah-ffi/releases) for
the full history of changes, including release notes and binary assets.
